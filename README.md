# 공동체 빌더스 (Community Builders)

안개에 갇힌 마을을 깨우는 사회정서교육(SEL) 3D 복셀 탐험 게임.

- **플레이**: https://cleveranawim-source.github.io/Community_builders/
- 16개 공동체 미션(마음알기·서로듣기·관계잇기·마을세우기) + 빛장벽 144개
- 반 코드 입력 시 반 전체 "마을 밝기" 공동 목표 + 교사 모니터링 화면(`#teacher`)
- 문항 수정: `src/data/quests.json`

## 개발

```bash
npm install
npm run dev        # http://localhost:5173
npm run community  # (선택) 같은 와이파이 아바타 공유 서버
npm run build      # dist/ 생성
```

수업 시연 팁: 주소 뒤 `?zoom=2`(확대) / `?zoom=0.55`(마을 전경)
